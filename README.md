# potatoMusic
An qt music player for linux

## dependencies 
- qt6
- qt6-multimedia
- libtag
## build
Run `build.sh`

#### **or**
```shell
qmake
make
./potatoMusic
```
## install
Run `make install` afrer building to install. `make uninstall` to uninstall
